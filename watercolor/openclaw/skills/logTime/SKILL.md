# logTime Skill                                                                                                      
                                                                                                                        
Append the current timestamp to `/tmp/logTime.log`.                                                                  
                                                                                                                        
## Usage                                                                                                             
                                                                                                                        
When the user says "log time" or similar, append the current timestamp to `/tmp/logTime.log`.                        
                                                                                                                       
## Script                                                                                                            
                                                                                                                        
Run `python3 scripts/logTime.py` to append the timestamp.

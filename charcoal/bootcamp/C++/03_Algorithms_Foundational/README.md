# Phase 3: Foundational Algorithms

## Key Techniques
1.  **Prefix Sums**: O(1) range sum queries after O(N) preprocessing.
2.  **Two Pointers**: Used for problems like "Sum of two elements" in a sorted array.
3.  **Sliding Window**: Efficiently processing subarrays of a fixed or variable size.
4.  **Binary Search on Answer**: Solving problems of type "Find the maximum X such that condition P(X) is true."

## Concepts to Master
- `pref[i] = pref[i-1] + a[i]`
- Inverting a condition for Binary Search: Is it monotonic?

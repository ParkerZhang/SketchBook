# Phase 1: C++ Fundamentals

## Key Concepts
1.  **Fast I/O**: Essential for competitive programming to prevent TLE (Time Limit Exceeded).
    ```cpp
    ios::sync_with_stdio(0); cin.tie(0);
    ```
2.  **Integer Overflow**: Always check constraints. Use `long long` for values exceeding $2 \cdot 10^9$.
3.  **Recursion**: Understanding base cases and recursive steps.
4.  **Pointers & References**: Crucial for building custom data structures.

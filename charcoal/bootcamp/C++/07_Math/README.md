# Phase 7: Number Theory & Mathematics

## Key Concepts
1.  **GCD / LCM**: Euclidean algorithm.
2.  **Primes**: Sieve of Eratosthenes ($O(N \log \log N)$).
3.  **Modular Arithmetic**: $(a+b)\%m, (a-b+m)\%m, (a \cdot b)\%m$.
4.  **Modular Inverse**: Fermat's Little Theorem ($a^{m-2} \% m$ if $m$ is prime).
5.  **Fast Exponentiation**: $a^b \% m$ in $O(\log b)$.
6.  **Combinatorics**: $nCr$ with modular inverse.

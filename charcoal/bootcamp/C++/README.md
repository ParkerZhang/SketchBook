# C++ Competitive Programming Bootcamp

This workspace is designed to guide you through mastering C++ for ACM-ICPC and other competitive programming contests.

## 🚀 Learning Path

### Phase 1: C++ Fundamentals & Competitive Setup
- **Directory**: `01_Basics/`
- **Topics**: Environment setup, Fast I/O, Data types, Control structures, Functions, Recursion, Pointers & References.

### Phase 2: Standard Template Library (STL) Mastery
- **Directory**: `02_STL/`
- **Topics**: Vectors, Deque, List, Stack, Queue, Priority Queue, Set, Map, STL Algorithms (`sort`, `binary_search`).

### Phase 3: Foundational Algorithms
- **Directory**: `03_Algorithms_Foundational/`
- **Topics**: Big O Notation, Prefix Sums, Two Pointers, Sliding Window, Binary Search (on answers), Greedy.

### Phase 4: Intermediate Data Structures
- **Directory**: `04_Data_Structures/`
- **Topics**: DSU, Fenwick Tree, Segment Tree, Sparse Table.

### Phase 5: Graph Theory
- **Directory**: `05_Graphs/`
- **Topics**: DFS/BFS, Dijkstra, Bellman-Ford, Floyd-Warshall, Kruskal/Prim, Topological Sort.

### Phase 6: Dynamic Programming (DP)
- **Directory**: `06_DP/`
- **Topics**: Knapsack, LCS, LIS, Range DP, Bitmask DP, Tree DP.

### Phase 7: Number Theory & Mathematics
- **Directory**: `07_Math/`
- **Topics**: GCD/LCM, Sieve of Eratosthenes, Modular Arithmetic, Fast Exponentiation, Combinatorics.

---

## 🛠 Setup Tips for Competitive Programming

1.  **Fast I/O**: Use `ios::sync_with_stdio(0); cin.tie(0);` at the start of `main()`.
2.  **Long Long**: Use `long long` for large numbers to avoid overflow.
3.  **Template**: Create a standard template in each directory to speed up development.

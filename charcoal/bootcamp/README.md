C++ Bootcamp (Charcoal)

Overview: A self-paced C++ bootcamp for experienced programmers transitioning to modern C++.

Modules:
- 00-setup: Tooling and build
- 01-basics: Syntax, types, control flow, functions
- 02-oo: Classes, RAII, smart pointers
- 03-stl: Containers, iterators, algorithms
- 04-modern-cpp: Move semantics, templates, concurrency

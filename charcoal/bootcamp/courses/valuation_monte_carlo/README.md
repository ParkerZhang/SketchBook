Course: Valuation via Monte Carlo & Numerical Methods

Overview:
Deep dive into Monte Carlo valuation, variance reduction, path-dependent payoffs, Greeks via pathwise and likelihood methods, and numerical analysis essentials.

Modules:
- Monte Carlo fundamentals and convergence
- Variance reduction techniques (antithetic, control variates)
- Greeks estimation and finite-difference vs pathwise
- Advanced topics: quasi-Monte Carlo, parallelization

Exercises: Monte Carlo pricer, variance reduction experiments, parallel implementation in C++.

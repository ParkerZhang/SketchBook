Course: Quantitative Finance with C++

Overview:
Hands-on quantitative finance topics implemented in modern C++: Black–Scholes, binomial trees, Monte Carlo simulation, calibration, numerical methods.

Modules:
- Mathematical foundations (probability, distributions)
- Black–Scholes analytic pricing
- Binomial trees and lattice methods
- Monte Carlo pricing and Greeks estimation
- Model calibration and optimization

Prerequisites: Comfortable with C++ basics, linear algebra, and basic probability.

Exercises: exercises/ under this course contain starter implementations and tests.
